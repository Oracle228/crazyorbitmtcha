# Orbit script (Matcha LuaVM + p UI)

Скрипт для [Matcha LuaVM](https://matcha-latte.gitbook.io/matcha). Орбит камеры вокруг выбранного игрока с хаотичной траекторией.

## Запуск

В Matcha выполни один главный файл:

```lua
-- Вставь содержимое matcha_orbit_main.lua или загрузи файл
```

Либо загрузи библиотеку p и основной скрипт по отдельности — в `matcha_orbit_main.lua` библиотека p уже подгружается по ссылке.

## Поведение

- **При старте:** персонаж телепортируется в «космос» `(123456789012, 123456789012, 123456789012)` и замораживается (не падает).
- **GUI (библиотека p):** вкладка Orbit → секция Players → у каждого игрока переключатель «Orbit: [имя]». Включение — камера переходит в режим орбиты вокруг этого игрока.
- **F1** — сворачивание/разворачивание меню (как в p).
- **Орбит:** камера движется вокруг цели с изменяющимся радиусом и углами, без ровной окружности — «рваная», но плавная траектория.
- **Когда HP цели становится 0:** орбит выключается, персонаж снова телепортируется в «космос» и замораживается.

## Файлы

- `matcha_orbit_main.lua` — основной скрипт (телепорт, заморозка, орбит, GUI, RunService).
- `matcha_orbit_chaos.lua` — отдельный модуль с формулой хаотичного смещения камеры (опционально, логика уже есть в main).

## Зависимости

- Matcha LuaVM (RunService, game, Players, workspace, Camera).
- Библиотека p подгружается из:
  `https://raw.githubusercontent.com/catowice/p/refs/heads/main/library.lua`

Если в Matcha нет глобальных `setrobloxinput`, `iskeypressed`, `isrbxactive`, которые использует p, меню может не реагировать на ввод — тогда их нужно замокать под свой исполнитель.
